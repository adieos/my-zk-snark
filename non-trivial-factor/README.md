this circuit is to prove that you know how to factor N but better, since now it also proves the factor is NOT 1 because that'd be trivial or useless

`input.json` would contain the factors of the public composite number `N` and must be kept secret. Also it would be way better if the factors are prime

`witness.wtns` is the witness file generated from `input.json` and also must be kept secret

`proof.json` is the legit proof generated by prover

`public.json` is basically the number `N` that is default

`v_key.json` is `Sv` used by verifier
